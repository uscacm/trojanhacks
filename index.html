Workshops

How to React and React Native

Learn React.js, a JavaScript library for building user interfaces on the web. Also learn how to use React Native to build mobile apps, and give your application a frontend!

How to Node

Learn how to build a backend using Node.js, the popular framework that executes JavaScript outside the browser. Learn the basics of Node and be building APIs in no time!

How to Google Cloud ML

Hear from a Google engineer on how to use machine learning in your project using Google Cloud products, and hear his experiences working on Google ML.

How to Electron

Create a desktop app effortlessly using Electron, built simply with JavaScript, HTML, and CSS. Learn how to turn any web app into a desktop app using this framework.
